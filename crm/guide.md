# باشگاه مشتریان — مستندات فرانت‌اند

---

## ساختار کلی

سکشن **باشگاه مشتریان** از دو بخش اصلی تشکیل شده:

1. **پرونده مشتریان** — مدیریت کامل مشتریان عادی و تجاری
2. **ارسال پیامک** — ارسال پیامک دسته‌جمعی به مشتریان عادی و تجاری تو دوتا تب جدا

---

## بخش اول — پرونده مشتریان

### چیدمان کلی

چیدمان این بخش دقیقاً مشابه بخش پرونده مشتریان در نسخه MVP است:

- **سمت چپ:** لیست مشتریان با سرچ‌بار
- **سمت راست:** جزئیات مشتری انتخاب‌شده (با کلیک روی هر مشتری)

---

### تب‌بندی

این بخش شامل دو تب است:

| تب | توضیح |
|---|---|
| مشتریان عادی | لیست مشتریانی که پروفایل تجاری ندارند |
| مشتریان تجاری | لیست مشتریانی که پروفایل B2B فعال دارند |

---

### پنل چپ — لیست مشتریان

**ویژگی‌ها:**
- سرچ‌بار بالای لیست
- سرچ از طریق query param روی همان endpoint لیست انجام می‌شود (endpoint جداگانه‌ای وجود ندارد)
- هر آیتم لیست شامل: نام کامل، شماره تلفن

**Endpoints:**

```
# مشتریان عادی
GET /customers/?search=...

# مشتریان تجاری
GET /customers/b2b/?search=...
```

---

### پنل راست — جزئیات مشتری

با کلیک روی هر مشتری، پنل راست سه بخش نمایش می‌دهد:

#### ۱. اطلاعات مشتری

فیلدها: نام کامل، شماره تلفن، آدرس، کد پستی، تصویر پروفایل

```
GET /customers/{id}/
```

#### ۲. پروفایل تجاری *(فقط در تب مشتریان تجاری به همراه جزعیات مشتری)*

فیلدها: نام تجاری، حداکثر بدهی مجاز، کد یونیک، درصد تخفیف، توضیحات

```
GET /customers/{customer_id}/b2b/detail/
```

#### ۳. لیست تراکنش‌ها

```
GET /customers/{customer_id}/transactions/
```

#### ۴. لیست فاکتورها

```
GET /customers/{customer_id}/invoices/
```

---

### عملیات CRUD

#### مشتری عادی

| عملیات | Method | Endpoint |
|---|---|---|
| افزودن | POST | `/customers/create/` |
| ویرایش | PATCH | `/customers/{id}/` |
| حذف | DELETE | `/customers/{id}/` |

#### پروفایل تجاری

| عملیات | Method | Endpoint |
|---|---|---|
| افزودن | POST | `/customers/{customer_id}/b2b/` |
| ویرایش | PATCH | `/customers/{customer_id}/b2b/detail/` |
| حذف | DELETE | `/customers/{customer_id}/b2b/detail/` |

> **نکته:** کلید شناسایی پروفایل تجاری همیشه `customer_id` است، نه `id` خود جدول B2B.

## بخش دوم — ارسال پیامک

### چیدمان

طراحی این بخش دقیقاً مشابه نسخه قبلی است فقط دوتا تب داره یکی برای مشتریجاری یکی برای مشتریان عادی.

### ویژگی‌ها

- **لیست مشتریان** با قابلیت انتخاب تکی
- **سرچ‌بار** برای جستجو در لیست
- **انتخاب همه** — دکمه‌ای برای انتخاب تمام مشتریان (مشابه نسخه قبلی)
- **فیلد متن پیامک** و دکمه ارسال

### رفتار

- انتخاب دونه‌دونه با چک‌باکس کنار هر مشتری
- انتخاب همه با چک‌باکس سربرگ لیست
- سرچ روی لیست نمایش‌داده‌شده فیلتر می‌کند

---

## نکات کلی

- تمام حذف‌ها **soft delete** هستند (مقدار `204 No Content` برمی‌گرداند)
- تمام اندپوینت ها فعلا باز هستن ولی پرمیدیشون با بخش کارمندان میاد
- فیلترهای مالی (تعداد تراکنش، جمع مبلغ و...) در صورت نیاز کارفرما روی لیست مشتریان در آینده اضافه خواهند شد
